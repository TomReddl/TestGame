Формат .xml-файла для хранения данных о книге:

<book> // обязательный оберточный тег
    <increaseSkill>1</increaseSkill> // необязательный тег, при первом прочтении книги увеличивается указанный в теге навык
    <type>note1</type> // обязательный тег, возможные значения: book, scroll, note, цифра- номер картинки-задника
    <lang>CATTGAR</lang> // обязательный тег, язык книги. Все языки перечислены в WorldLangEnum.java
    <pages>
        <page>text1</page>
        <page>text2</page> // Для типа book может быть больше 1 страницы, для note может быть картинка и одна страница, для scroll всегда должна быть одна страница
        <page>IMG_image_name</page> // картинка должна начинаться с префикса "IMG_". В папке с картинками для книг могут лежать разные изображения для разных языков, например, image_name_ru.png
    </pages>
</book>
