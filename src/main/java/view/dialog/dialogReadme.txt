1. Условия видимости ответов в диалоге:

"playerHasItem ID_предмета" - есть ли в инвентаре у персонажа предмет с id = ID_предмета

2. Условия для выбора следующей фразы в диалоге:

"playerHasItem ID_предмета ID_следующей_фразы else ID_другой_фразы" - если у персонажа есть в инвентаре предмет с id = ID_предмета, будет осуществлен переход на фразу с ID_следующей_фразы, иначе- на фразу с ID_другой_фразы

3. Замена части текста в фразе или ответе:

"%playerName" - имя игрока
"%NPCName" - имя NPC, с которым идет диалог

4. Скрипты при выборе фраз в диалоге:

"playerAddItem ID_предмета Количество" - добавить в инвентарь игроку указанное количество предметов с id = ID_предмета
